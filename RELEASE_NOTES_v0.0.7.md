# Sovereign GE v0.0.7 — P2P multi-device sync + encrypted backup

Sovereign GE keeps your documents, contacts, messages, and an on-device AI on
your own machine — local-first, no cloud. v0.0.7 makes it work across more than
one of your machines, privately.

## Multi-device sync (peer-to-peer)

Sync your workspace across your paired devices over an encrypted peer-to-peer
channel — no cloud, no server in the middle:

- Documents, contacts, messages, conversations, relationships, and suggested links.
- An interactive device-pairing handshake — pair two devices directly.
- Encrypted peer-to-peer backup with social-recovery key splitting, so you can
  recover your data with the help of people you trust rather than a company.

## Security & stability

Security and stability hardening across the workspace.

## Install — Windows (NVIDIA / CUDA)

1. Download `sovereign-ge-v0.0.7-windows-x64-cuda.zip`.
2. Unzip it (keep the files together).
3. Run `sovereign.exe`.

Requires an NVIDIA GPU with a recent driver. The CUDA runtime is bundled in the
zip — no CUDA toolkit install needed.
