Important License Notes

SurrealDB (Business Source License)

SurrealDB is distributed under the Business Source License (BSL-1.1), which is a source-available license. It automatically converts to Apache 2.0 four years after release. For Sovereign OS purposes (open-source, non-commercial), this is compatible with AGPL-3.0. See SurrealDB license for details.

Llama 3.1 Community License

Llama 3.1 models are distributed under the Llama 3.1 Community License. Personal use is permitted. Production use may require additional agreements with Meta. See Meta's license terms.

CUDA Toolkit (Optional, Proprietary)

CUDA Toolkit is optional and proprietary software from NVIDIA. It is not required to run Sovereign OS. GPU acceleration is optional and requires agreeing to NVIDIA's license terms separately.

LGPL Components

GTK 4 and GStreamer are licensed under LGPL. As a work-around for AGPL-3.0 compatibility, we dynamically link these libraries. LGPL requirements (provision of object files for relinking) are satisfied via open-source distribution of Sovereign OS.

How to Verify Licenses

To verify all licenses in the Sovereign OS codebase, run:

# Generate license report for Rust crates
cargo-license

# Verify vendored dependencies
ls -la vendor/

# Check Python dependencies
pip list --format=json

# View Cargo.lock for pinned versions
cat Cargo.lock
