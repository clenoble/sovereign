Nature of the Project

Sovereign OS is a distributed, open-source software project. It is not a hosted service. Key characteristics:

  • No servers operated: The Sovereign OS project does not operate or maintain any servers.
  • No user data collection: The software does not send data to any service operated by the project.
  • No accounts maintained: There is no account system operated by the Sovereign OS project.
  • No capability to intercept, access, or modify user data: By design, the project has no technical means to access, intercept, modify, or delete user data stored on your device.
  • Decentralized: All data is stored locally or synchronized peer-to-peer (if enabled) without central storage.
What this means: If you are concerned about data privacy and control, Sovereign OS cannot access your data by design. We have no ability to surveil, audit, recover, or restore your information.
